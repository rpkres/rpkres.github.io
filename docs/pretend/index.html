Pretend

Pretend is a story creation tool for rapid development of high quality movies and videos.
 

MORE

I was heavily involved in the user interface (UI) and user experience (UX) design for the product.

My role included design mockups, creation of application assets utilizing Sketch (Bohemian Software) and implementation of user facing elements utilizing Apple’s application construction tool, Xcode.

Pretend Application (MacOS)

Pretend on MacOS combines scene based, screenwriting and movie making tools.
 

More

There is no prescribed method to building projects in Pretend. A director can work in any way they are comfortable with.

Beginning in the screenplay editor can help quickly build an overall blueprint for a project.

Productions such as music videos or commercials don’t require a full movie script, but the screenplay editor can be used to help create an animatic or shooting script.

The screenplay editor can also aid in creating line-ups for planning camera positions and angles for a shoot.

Video clips are added to the project library and associated with scenes in the screenplay.

 

Library items are added to the project timeline, which can handle editing, compound video clips, audio, color correction and effects.

Arcs can be used as traditional story arcs, and are also available to visualize aspects such as choosing camera angles, wardrobe, color grading and soundtracks.

Finished videos can be saved to disk and shared on sites such as YouTube. An EDL (Edit Display List) can be exported for use in other editing packages.

Pretend Assistant (iOS)


Pretend includes an iOS helper application named Pretend Assistant (PA).
 

More

PA is useful for adding reference videos and photos to a Pretend project. It’s invaluable for location scouting.

I used the same design principals and tools for PA as the Pretend application.

Sketch (Bohemian Software) was used for mockups and asset generation along with Xcode (Apple) for final layout and constraints.

Pretend Logo

More

Pretend integrates screenwriting and movie making tools so I based the logo design on an old typewriter key surrounded by stylized movie film sprockets (perforations).

 

MacOS Integrated Icons

I included a suite of MacOS styled icons to integrate the appearance of Pretend with Apple’s Finder.
